Multiline Equations

  1. MathSpeak
    2. This is the regular MathSpeak rendering with standard walking of the expression.
  2. Simplified speech
    3. This drastically simplifies the speech output for the overall table, omitting much of the layout announcements of MathSpeak.
  3. Mathspeak with initial component repeated
    4. This simply repeats the omitted first element in the first column (indicated by a thus). The actual voicing can be ameliorated, but the main question is: Is that useful at all?
  4. MathSpeak and Table Walker
    5. In Table walker mode, shift arrow keys allows to walk vertically and horizontally on the single cells of a table. Currently the speech on the cells is the same as in regular walking mode. However there are a couple of questions regarding this.

    Points of interest: Is the walker useful at all?

    Currently only the column number is announced when walking on cells. But Would it be useful to announce the full coordinates of the next position (i.e row and column), or no coordinates at all?

Multiline Equations with Labels

All examples are with Mathspeak. Style is either Verbose, Brief or SuperBrief, as indicated.

  1. Verbose Label
    2. Verbose label that announces it's start and end.
  2. Brief Label
    3. Brief label that announces it's start only. Since the start of the row with it's number is announced anyway, this might be sufficient.
  3. Super Brief Label
    4. Superbrief label that announces it's start only and omits as much as possible parenthesis.

    Note that labels can not explicitly explored. It would be interesting to see if the testers express a desire to explore labels. In particular, in the last of the four examples it might be helpful for the reader to go through details.
